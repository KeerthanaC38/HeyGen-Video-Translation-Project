# Setup and Run Instructions

1. **Clone the Repository**

   ```bash
   git clone <repository-url>
   cd <repository-directory>
   ```
   
2. **Build the project**
    ```bash
   mvn clean install
   ```
   
3. **Run the Application**
    ```bash
   mvn spring-boot:run
    ```
   
4. **Access the Application**: 
   - The server will be running at http://localhost:8080.
